# StockAI - 轻量级股票分析平台

## 项目简介
简洁的 AI 股票分析服务，用户付费使用平台 AI 能力。

## 技术栈
- 后端: FastAPI
- 前端: Vue3 + Element Plus
- 数据库: MongoDB
- AI: DeepSeek API

## 核心功能
1. 用户注册/登录
2. 会员系统 (免费/付费)
3. AI 股票分析
4. 每日次数限制

## 部署
```bash
docker-compose up -d
```

## 目录结构
```
StockAI/
├── backend/          # FastAPI 后端
├── frontend/         # Vue3 前端
└── docker-compose.yml
```
